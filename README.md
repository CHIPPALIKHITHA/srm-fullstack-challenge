# SRM Full Stack Engineering Challenge

This project is built for the SRM Full Stack Engineering Challenge.

## Tech Stack
- Node.js
- Express.js
- JavaScript
- HTML
- CSS
- REST API

## API Endpoint

POST /bfhl

## Features
- Validates node format
- Detects invalid entries
- Detects duplicate edges
- Builds hierarchy trees
- Detects cycles
- Calculates depth
- Returns summary object
- Frontend UI to test API

## Run Backend

```bash
cd backend
npm install
npm start
